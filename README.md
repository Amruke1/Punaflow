# Punaflow

Punaflow is a workforce management platform designed to connect workers with companies through structured profiles, communication, and project coordination.

## Features

- Worker and Company profiles
- Portfolio management for workers
- Job request and hiring system
- Messaging system between users
- Contract agreement structure
- Basic scheduling and availability

## Architecture

The project follows a layered architecture:

- **Models** ? Data structures
- **Services** ? Business logic
- **Data** ? Repository pattern and data handling
- **UI** ? Console interface

### Patterns Used

- Repository Pattern
- Separation of Concerns
- Layered Architecture

## Technologies

- C#
- .NET
- File-based data storage (CSV)
- Git & GitHub
